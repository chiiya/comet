Aral Station API

The Aral Station API can be used to retrieve information on everything related to Aral gas stations. This includes requesting details for specific stations, searching for stations by location or various filters, as well as up-to-date pricing details.

Versioning

Currently, the only supported version of the API is v2, accessible under /api/v2/.

JSON API

In the following section, a short overview of the general capabilities of the API will be given.

Note: This documentation will display parameters with brackets ([ and ]) for readability, but actual URLs will need to be percent-encoded (%5B and %5D).

Filtering and Search

Filtering lets you query data that contains certain matching attributes or relationships. These take the form of filter[attribute]=value. For example, you can request all the stations offering a selection of services:

/stations?filter[services]=petit_bistro,aral_store,opening_24

Pagination

Pagination is enabled by default for large sets of results (as is the case with stations). You can optionally disable pagination by passing along the paginated parameter and setting it to false:

/stations?paginated=false

Paginated responses will include URLs for the first, last, previous and next page of resources in the links object. Some meta information regarding pagination and result count is also included:

"links": {
    "first": "http://local.api.tankstelle.aral.de/api/v2/stations?page=1",
    "last": "http://local.api.tankstelle.aral.de/api/v2/stations?page=154",
    "prev": null,
    "next": "http://local.api.tankstelle.aral.de/api/v2/stations?page=2"
},
"meta": {
    "current_page": 1,
    "from": 1,
    "last_page": 154,
    "path": "http://local.api.tankstelle.aral.de/api/v2/stations",
    "per_page": 15,
    "to": 15,
    "total": 2300
}

Includes

For some resources (e.g. stations), you can also include related resources with include=[relationship]. A comma-delimited list can be used to request multiple relationships.

/stations?include=brand,cards,services

Dates

All dates will be UTC by default. If you wish to use a different timezone, you should convert these dates locally on your client.

Filters

Filter Example Notes
mail heinz.gusenbauer@tankstelle.de Filter by email address
station_id 14011500,16084700,14107400 Filter by stations IDs. Only stations that match one of the comma separated IDs will be included in the response
post_code 14163 Filter by post code (stations must be within this post code)
city Berlin Filter by city (stations must be within this city)
services petit_bistro,superwash,opening_24 Filter by services (stations must offer all the listed services)
address Vogelsanger Str 66 Filter by address (will do a search within a given distance around the address)
latitude 51.199320 Filter by latitude (only use in combination with the longitude filter to search by geo coordinates)
longitude 7.075124 Filter by longitude (only use in combination with the latitude filter to search by geo coordinates)
polyline {_wuH_jni@dmF_tH Filter by a given Google polyline
distance 1 Maximum distance (km) for geo polyline filtering (default 1)
brand 1,3 Filter by brand ids. Only stations that belong to one of the comma separated brand ids will be included in the response
channel 1,3,7,10 Filter by distribution channel ids. Only stations that belong to one of the comma separated channel ids will be included in the response
preset aral_b2c Applies some pre-defined filters. One of [aral_b2c, aral_b2b, hessol, gasolin]

Filter Presets

Filter presets apply some pre-defined filters to your query. These filters cannot be overwritten, but they can be extended. This means that if a filter preset applies the brand and channel filter, you cannot overwrite the values for those filters, however you are free to extend the query with additional filters (such as services or post_code).

Preset Applied filters Explanation
aral_b2c brand, channel Only stations with brand ARAL Tankstelle and a distribution channel other than UNCO are returned.
aral_b2b brand, channel Only stations with brand ARAL Tankstelle and a distribution channel other than UNCO are returned.
hessol brand Only stations with brand Hessol Tankstelle are returned.
gasolin brand Only stations with brand Gasolin Tankstelle are returned.
